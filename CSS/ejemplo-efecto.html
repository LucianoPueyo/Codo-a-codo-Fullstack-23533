Explicación

- La caja mascara tiene un ancho de 100px, el mismo tamaño que las imagenes de color rojo y verde. Tiene overflow hidden para ocultar los excesos y un border radius elevado para que funcione como una mascara circular.
- La caja contenedor tiene un ancho fijo igual a la suma de todas las imagenes, en nuestro caso 200px porque tenemos una roja y una verde, cada una de 100px. Esta caja contenedor tiene un position relative con left: 0 por defecto. cuando hacemos un hover (posicionamos el mouse sobre la caja) dicha propiedad cambia a left: -100px, de esta forma desplazamos la caja contenedora hacia la izquierda lo mismo que mide una caja, haciendo que la roja se haga invisible y la verde aparezca. El segundo ejemplo además tiene un efecto de transicion.
- Las cajas roja y verde tienen display inline-block para poder posicionarse una a la derecha de la otra. Tienen un tamaño fijo de 100x100px y text-align center para que el texto este centrado. Al darle un line-height de 100px (igual a la altura de la caja) podemos lograr un efecto rapido de centralizado vertical ( mas adelante veremos como hacer esto con flexbox).
